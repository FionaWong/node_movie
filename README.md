#通过node构建电影网站与后台管理

***
##分2个版本
1. 简单版本simple(完成)
2. 复杂版本complex(正在编写)
***

simple
-------------


######安装模依赖
    npm install
    bower install
######启动文件
    node app.js


> ## 一共有四个路由地址
> 
> 1.   index
> 2.   admin/input
> 3.   admin/list
> 4.   admin/detail
> 
> 开始访问：
> 
>     localhost:3000
